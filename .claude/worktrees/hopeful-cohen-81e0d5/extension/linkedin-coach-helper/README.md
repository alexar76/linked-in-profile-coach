# Profile Coach — LinkedIn copy helper

Chrome extension: copies the visible LinkedIn page text with a section header (`EXPERIENCE:`, etc.) for paste into Profile Coach.

## Install (developer mode)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. **Load unpacked** → select this folder
4. On linkedin.com, open a profile section and click the extension → **Copy for Profile Coach**
5. In Profile Coach → LinkedIn → **Import from clipboard**
